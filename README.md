# Dashboard de E-commerce

Este projeto apresenta um dashboard interativo criado para monitorar e analisar os principais indicadores de performance em um e-commerce. Desenvolvido com Power BI, o dashboard oferece insights valiosos para otimizar as vendas, identificar produtos de destaque e avaliar o desempenho por canais de vendas.

## 🛠️ Descrição do Projeto

O **Dashboard de E-commerce** fornece uma interface amigável e rica em dados para acompanhar o faturamento, volume de vendas, categorias de produtos mais vendidas e representatividade dos canais (website e mobile). Este projeto ajuda os gestores a tomar decisões baseadas em dados para alcançar melhores resultados.

### **Funcionalidades**

- **Visão Geral do Faturamento**  
  Acompanhe o total de faturamento, quantidade de itens vendidos, número de pedidos e clientes.
  
- **Análise de Desempenho por Produto**  
  Visualize o valor total gerado por cada categoria e produto.

- **Representatividade por Canal de Venda**  
  Compare as vendas realizadas por website e mobile app.

- **Tendências de Vendas**  
  Identifique picos e sazonalidades ao longo do tempo.

- **Filtros Interativos**  
  Aplique filtros para visualizar informações por produto, categoria ou intervalo de datas.
